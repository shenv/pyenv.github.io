shenv's mirror for shell archives

Welcome to the shenv's official mirror for shell archives.

The shell-build plugin is a shenv plugin that allows users to install shells from source. It uses archives available at the shells sub-URL for faster downloads.

You can enforce shell-build to use optional mirror site with overriding SHELL_BUILD_MIRROR_URL.


$ export SHELL_BUILD_MIRROR_URL="http://yoursite.example.com/shells"
$ shenv install bash-4.4.12

The filenames of the archives must be the sha256sum of its content. The following links are the contents of the shenv's official mirror for shell archives.
